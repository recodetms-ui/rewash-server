# 🚀 تثبيت خادم RE-WASH على الـ VPS

دليل خطوة بخطوة لـ VPS بنظام Ubuntu (مثل خادمك على Hostinger: `187.127.77.39`).
نفّذ الأوامر بالترتيب عبر SSH. كل الأوامر مُختبَرة.

> الخادم نفسه مُختبَر بالكامل (١١ اختبار نجحت): تسجيل أجهزة، تراخيص، مزامنة، نسخ احتياطي،
> لوحة تحكم، وإيقاف عن بُعد.

---

## ١) الاتصال بالخادم
من جهازك:
```bash
ssh root@187.127.77.39
```

## ٢) تثبيت Node.js 20
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs
node -v    # لازم يطلع v20.x
```

## ٣) تثبيت PostgreSQL
```bash
apt-get install -y postgresql postgresql-contrib
systemctl enable --now postgresql
```
أنشئ قاعدة البيانات والمستخدم (بدّل كلمة السر بكلمة قوية):
```bash
sudo -u postgres psql -c "CREATE USER rewash WITH PASSWORD 'كلمة_سر_قوية_جداً';"
sudo -u postgres psql -c "CREATE DATABASE rewash_db OWNER rewash;"
```

## ٤) رفع كود الخادم
انسخ مجلد `rewash-server` للخادم (عبر SCP أو Git). مثال بـ SCP من جهازك:
```bash
scp -r rewash-server root@187.127.77.39:/opt/
```
ثم على الخادم:
```bash
cd /opt/rewash-server
npm install --omit=dev
```

## ٥) إعداد ملف البيئة (.env)
```bash
cp .env.example .env
nano .env
```
املأ القيم. ولّد أسرار عشوائية قوية بهذا الأمر (شغّله 3 مرات وخذ كل ناتج):
```bash
node -e "console.log(require('crypto').randomBytes(48).toString('hex'))"
```
- `DATABASE_URL` → ضع كلمة سر postgres اللي اخترتها.
- `JWT_SECRET` → ناتج عشوائي.
- `ADMIN_API_KEY` → ناتج عشوائي (هذا مفتاحك السري للإدارة).
- `LICENSE_PUBLIC_KEY` → خليه كما هو (نفس مفتاح برنامجك).
- `ALLOWED_ORIGINS` → `https://recode-systems.com,https://www.recode-systems.com`

## ٦) إنشاء الجداول
```bash
npm run migrate
```
لازم يطلع: `✅ اكتمل الترحيل.`

## ٧) تشغيل الخادم دائماً (pm2)
```bash
npm install -g pm2
pm2 start src/index.js --name rewash-server
pm2 save
pm2 startup    # نفّذ الأمر اللي يطلعه لك
```
تأكد:
```bash
curl http://localhost:3000/api/health
```
لازم يرجّع: `{"ok":true,...}`

## ٨) ربط الدومين + HTTPS (Nginx + SSL مجاني)
```bash
apt-get install -y nginx certbot python3-certbot-nginx
```
أنشئ إعداد Nginx:
```bash
nano /etc/nginx/sites-available/rewash
```
الصق (بدّل الدومين إذا حبيت subdomain مثل api.recode-systems.com):
```nginx
server {
    server_name recode-systems.com www.recode-systems.com;
    client_max_body_size 12M;
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```
فعّل واحصل على SSL:
```bash
ln -s /etc/nginx/sites-available/rewash /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx
certbot --nginx -d recode-systems.com -d www.recode-systems.com
```
> ملاحظة: لازم الدومين `recode-systems.com` يكون موجّه (DNS A record) لـ `187.127.77.39` قبل certbot.

بعدها خادمك متاح على: `https://recode-systems.com/api/health`

## ٩) جدار الحماية (اختياري لكن مُستحسن)
```bash
ufw allow OpenSSH
ufw allow 'Nginx Full'
ufw enable
```

## شنو بعد التثبيت؟
الخادم جاهز. الخطوات الجاية (أبنيها لك):
1. **لوحة التحكم PWA** — واجهة المالك (تُوضع في مجلد `public/` ويخدمها نفس الخادم).
2. **عميل المزامنة في تطبيق الكاشير** — كود بسيط يضاف لـ Electron يرفع المبيعات/النسخ
   الاحتياطية للخادم لما يتوفر نت (يبقى التطبيق يشتغل أوفلاين بالكامل).

أي خطأ يطلع وقت التثبيت، انسخه لي وأحله.
